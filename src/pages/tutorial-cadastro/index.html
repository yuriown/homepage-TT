Tutorial: Cadastro de Clientes no Tiny

Este tutorial irá guiá-lo passo a passo no processo de cadastro de clientes no sistema Tiny.

Passo 1: Identificar o CNPJ do cliente

Certifique-se de que você tem o CNPJ correto do cliente antes de iniciar o processo.

Passo 2: Buscar o CNPJ

Utilize os sites da Receita Federal, SINTEGRA ou Simples Nacional para consultar o CNPJ do cliente e validar os dados.

Passo 3: Acessar o Tiny

No sistema Tiny, vá para o menu Cadastros e selecione Clientes e Fornecedores.

Passo 4: Incluir Cadastro

Clique no botão de Incluir Cadastro para adicionar um novo cliente.

Passo 5: Inserir CNPJ e UF

Insira o CNPJ e a UF do cliente. O sistema pode preencher automaticamente alguns campos, caso contrário, preencha manualmente.

Passo 6: Verificar Contribuinte de ICMS

Verifique se o cliente é contribuinte de ICMS utilizando esta tabela de contribuintes.

Passo 7: Verificar Simples Nacional

Verifique se o cliente é do Simples Nacional e preencha as informações nos dados complementares.

Passo 8: Inserir CNAEs nas Observações

Busque todos os CNAEs do cliente no site da Receita Federal e insira no link de observações para transformá-los e incluir no cadastro.

Passo 9: Acessar CRM no Tiny

Vá para o menu Vendas no Tiny e selecione CRM.

Passo 10: Incluir Assunto

Adicione um novo assunto relacionado ao cliente.

Passo 11: Inserir CNPJ na caixa "Cliente"

Insira o CNPJ do cliente na caixa de texto correspondente no CRM.

Passo 12: Inserir Origem do Cliente

Indique de onde o cliente veio, seguindo o padrão definido no Keep.

Passo 13: Inserir Produto e Dados da Venda

Informe o produto que o cliente deseja, a data atual e o vendedor responsável pela venda.
